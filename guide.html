소개

이 앱이 뭔가요?

🛒

간편한 빵 주문 앱

교회 내 빵집에서 판매하는 빵을 스마트폰으로 손쉽게 주문할 수 있는 앱입니다. 원하는 빵을 골라 수량을 선택하고, 계좌로 입금하면 주문 완료!

📅

주문 가능 요일

빵 배송 주문은 매주 월 · 화 · 금 · 토요일에만 가능합니다.
계좌 입금 확인 후 주문이 최종 승인됩니다.

STEP 1

회원가입

1

로그인 화면 하단 회원가입 선택

처음 이용하시는 분은 로그인 화면 하단의 회원가입 버튼을 눌러주세요.

2

정보 입력

이름(닉네임), 전화번호, 이메일, 비밀번호를 입력합니다.
비밀번호는 4자리 이상으로 설정해 주세요.

3

약관 동의 후 가입 완료

서비스 이용약관 + 개인정보 수집·이용 동의 체크 후 가입 완료!

✅ 전체 동의 버튼으로 한 번에 체크 가능
STEP 2

주문하기

1

하단 🛒 주문하기 탭 선택

로그인 후 하단 메뉴에서 주문하기 탭을 누르면 오늘의 빵 목록이 나타납니다.

2

빵 선택 및 수량 조정

+ / − 버튼으로 수량을 선택합니다. 여러 종류를 함께 담을 수 있어요.

3

장바구니 확인 후 주문

하단 장바구니 버튼 → 상품 확인 → 픽업 장소 선택 → 주문하기 클릭!

4

계좌 입금

주문 완료 화면에 표시된 계좌번호로 합계 금액을 입금하면 끝!

💳 입금 확인 후 최종 승인됩니다
STEP 3

주문 내역 확인

📄

주문 상태 확인

하단 메뉴 주문내역 탭에서 내 주문 현황을 확인할 수 있습니다.

🕐 주문 중 — 입금 대기 중
✅ 주문 완료 — 판매자 승인됨
🚚 배송 중 — 배송 진행 중
🏁 배송완료 — 수령 완료
🗑️

주문 취소

판매자 승인 전에만 주문 내역에서 직접 취소할 수 있습니다.

📅

날짜로 조회

주문 내역 상단의 날짜 필터로 원하는 기간의 주문을 조회할 수 있습니다.

판매자 전용

판매자 기능

📋

주문 관리

들어온 주문을 승인하거나 취소할 수 있습니다. 날짜 필터(기본: 어제~오늘)로 빠르게 조회 가능합니다.

🍞

상품 관리

빵 상품을 추가·수정·삭제할 수 있습니다. 사진, 상품명, 가격을 자유롭게 편집 가능합니다.

👥

회원 관리

가입 회원을 조회하고 비밀번호 초기화(1234) 또는 강제 탈퇴 처리할 수 있습니다.

⚙️

설정

입금 계좌 정보와 관리자 비밀번호를 변경할 수 있습니다.
