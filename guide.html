소개

이 앱이 뭔가요?

🛒

간편한 빵 주문 앱

교회 내 빵집에서 판매하는 빵을 스마트폰으로 손쉽게 주문할 수 있는 앱입니다. 원하는 빵을 골라 수량을 선택하고, 계좌로 입금하면 주문 완료!

📅

주문 가능 요일

빵 배송 주문은 매주 월 · 화 · 금 · 토요일에만 가능합니다. 계좌 입금 확인 후 주문이 최종 승인됩니다.

💜

포인트 적립 혜택

주문 승인 시 실결제금액의 5%가 자동 적립됩니다. 10,000P 이상 보유 시 결제에 사용할 수 있어요!

STEP 1

회원가입

1

로그인 화면 하단 회원가입 버튼 클릭

처음 이용하시는 분은 로그인 화면 하단의 회원가입 버튼을 눌러주세요.

2

정보 입력

이름(닉네임), 전화번호, 이메일, 비밀번호를 입력합니다. 비밀번호는 4자리 이상으로 설정해 주세요.

3

필수 약관 동의 후 가입 완료

서비스 이용약관 + 개인정보 수집·이용 동의 체크 후 가입하기를 누르면 완료!

✅ 전체 동의 버튼으로 한 번에 체크 가능
STEP 2

주문하기

1

하단 🛒 주문하기 탭 선택

로그인 후 하단 메뉴에서 주문하기 탭을 누르면 오늘의 빵 목록이 나타납니다.

2

빵 선택 및 수량 조정

+ / − 버튼으로 원하는 빵의 수량을 선택합니다. 여러 종류를 함께 담을 수 있어요.

3

포인트 사용 여부 결정

장바구니 하단의 💜 포인트 사용 칸에서 보유 포인트를 확인하고, 사용할 금액을 입력하세요.

💜 10,000P 이상 보유 시 전부 또는 일부 사용 가능
4

수령 장소 선택 후 주문하기

픽업 장소를 선택하고 주문하기 버튼을 클릭합니다. 주문 금액 / 포인트 차감 / 실결제금액이 한눈에 표시됩니다.

5

계좌 입금

주문 내역 화면에 표시된 계좌번호로 실결제금액을 입금하면 완료! (포인트 차감 후 금액)

💳 입금 확인 후 판매자가 최종 승인합니다
STEP 3

주문 내역 확인

📄

주문 상태 확인

하단 주문내역 탭에서 내 주문의 현재 상태를 확인할 수 있습니다.

🕐 주문 중 — 입금 대기 중 (취소 가능)
✅ 주문 완료 — 판매자 승인 + 포인트 적립
🚚 배송 중 — 배송 진행 중
🏁 배송완료 — 수령 완료
📅

날짜로 조회

상단 날짜 필터(기본: 어제~오늘)로 원하는 기간의 주문을 조회할 수 있습니다.

🗑️

주문 취소

판매자 승인 전(주문 중) 상태에만 주문 취소 버튼이 표시됩니다. 포인트로 결제한 경우 포인트는 즉시 차감되므로 신중하게 주문해 주세요.

포인트

포인트 시스템

💜 적립 방법

판매자가 주문을 승인할 때 실결제금액(포인트 차감 후)의 5%가 자동으로 적립됩니다.

💜 사용 방법

보유 포인트가 10,000P 이상일 때 주문하기 화면에서 사용 가능합니다.
원하는 금액만큼 부분 사용하거나 전액 사용할 수 있습니다.

💜 포인트 확인

하단 내 계정 탭에서 현재 보유 포인트 잔액을 언제든지 확인할 수 있습니다.

판매자

판매자 기능 안내

📋

주문 관리

접수된 주문을 승인하거나 취소할 수 있습니다. 날짜 필터(기본: 어제~오늘)로 빠르게 조회하고, 승인 시 자동으로 5% 포인트가 적립됩니다.

🍞

상품 관리

판매할 빵 상품을 추가·수정·삭제합니다. 사진, 상품명, 가격을 언제든지 편집할 수 있습니다.

👥

회원 관리

가입 회원 조회 및 포인트 잔액 확인, 비밀번호 초기화(1234), 강제 탈퇴 처리가 가능합니다.

💜

포인트 이관

회원 간 포인트를 직접 이관할 수 있습니다. 자동이관 설정을 사용하면 특정 회원이 포인트 적립 시 지정한 다른 회원에게 자동으로 이관됩니다.

⚙️

설정

입금 계좌 정보와 관리자 비밀번호를 변경할 수 있습니다.

주의
