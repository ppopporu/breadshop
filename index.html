🔥 Firebase Firestore 무료 영구 DB

아래 4단계를 따라 무료 DB를 연결하세요
신용카드 없이 영구 무료 (Spark 플랜)

⚠️ Firebase SDK 로드 실패

인터넷 연결을 확인하거나 페이지를 새로고침(F5)해주세요.

1
Firebase 프로젝트 생성
console.firebase.google.com 접속
→ 구글 계정 로그인 → 프로젝트 만들기
→ 이름 입력 (예: breadshop) → 계속 → 완료
2
Firestore 데이터베이스 생성
왼쪽 메뉴 Firestore Database 클릭
→ 데이터베이스 만들기 → 테스트 모드 선택
→ 위치: asia-northeast3 (Seoul) → 완료
3
보안 규칙 설정 (영구 허용)
Firestore → 규칙 탭 → 아래 코드로 교체 → 게시
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if true; } } }
4
앱 설정 코드 붙여넣기
⚙️ 프로젝트 설정 → 내 앱 → </> 웹 클릭
→ 닉네임 입력 후 앱 등록
→ firebaseConfig 객체 전체 복사
→ 아래 입력란에 붙여넣기
